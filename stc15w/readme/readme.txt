3，红外发送接收原理
	红外传输NEC协议
	载波38KHZ，逻辑1为2.25ms，脉冲时间560us；逻辑0为1.12ms,脉冲时间为560.
	
	协议格式
	(1),首先发送9ms的高电平信号
	(2),然后发送4.5ms的低电平信号
	(3),接下来发送8bit的地址码(低位在前)
	(4),然后是8bit的地址反码,用于检验地址是否出错
	(5),接下来是8bit的命令码(低位在前)
	(6),然后是8bit的命令反码,用于检验命令码是否出错

	重复码
	如果一直按着按键，将以110ms的周期发送重复码，重复码由9ms高电平，2.25ms低电平及560us高电平组成。
